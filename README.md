# 🏆 Quiz League of Legends: Qual Campeão Você é? 🎮

Bem-vindo ao **Quiz League of Legends**! 🧠✨  
Descubra qual campeão do *League of Legends* mais combina com você com base nas respostas que você dá às perguntas do quiz!

## 🔥 Sobre o Projeto
Este quiz foi desenvolvido para analisar sua personalidade e estilo de jogo, comparando suas respostas às características dos campeões do *League of Legends*. No final, você descobrirá qual campeão representa você no universo de Runeterra!

## 🚀 Funcionalidades
- 🔹 Perguntas interativas e personalizadas.
- 🔹 Resultados baseados em características dos campeões do jogo.
- 🔹 Interface amigável e dinâmica.
